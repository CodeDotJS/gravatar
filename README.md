> Download profile picture and extract information of any gravatar user by it's email.

## Install

```
$ npm install --global gravatar-of

```

## Usage

```
$ image-of -u /rushforlinux@gmail.com -n rishi


  Usage

    $ gravatar-of

              Options:

                 -u    Email ID 							 [required]

                 -n    save image as [ unicorn ]             [required]
```
